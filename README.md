# Concert Aid

##### Android App To Find Local Concert Suggestions Based Off An Entered Artist, 4/22/2016

#### By Kyle Derrick

## Description

This Android applicaion will allow users to enter an artist and receive a list of suggested upcoming concerts taking place in the selected city. Users will also be able to save a list of concerts that they are interested in.

## Setup/Installation Requirements
You will need the following programs installed on your computer.
* Android Studio
* Java JDK 8+
* Android SDK

### To Run
* In a terminal window, navigate to ~/AndroidStudioProjects
* Run `git clone https://github.com/kderrick/concertaid.git`
* Navigate to ~/AndroidStudioProjects/ConcertAid
* Run on either an emulator or an Android OS Device connected to a computer

##### To set up an emulator
* Select Run > Run 'app'
* Click 'Create New Emulator'
* Select the device you would like to emulate (Recommended: Nexus 6)
* Select the API level you would like to run - click 'Download' if not available (Recommended: Marshmallow - ABI: x86)
* Select configuration settings for emulator
* Click 'Finish' and allow Emulator to run
